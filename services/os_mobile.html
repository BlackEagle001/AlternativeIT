Les systèmes d'exploitation pour mobiles

Les smartphones possèdent aussi des systèmes d'exploitation, réalisant la même tache que ceux pour ordinateur personnel.

Avec les systèmes d'exploitation mobiles, le terme rom custom est souvent utilisé. Une rom est un OS qui a subi des transformations afin d'avoir un "look" différent mais en gardant la même base. Customisée signifie que les améliorations n'ont pas été faites par des producteurs mais plutot par la communauté.

LineageOS

LineageOS est une rom custom Android open-source et gratuite.
Il est compatible sur tablette, smartphone et ordinateur. Il se présente comme le successeur de CyanogenMod.
C'est l'une des rom custom les plus utilisées.

Replicant

Replicant est un système d’exploitation pour mobile qui se base sur LineageOS.
Il s'agit d'un OS gratuit pour les smartphones et tablettes.
Il vise à proposer une alternative entièrement libre.

CopperheadOS

CopperheadOS est un OS basé sur la plateforme mobile Android et plus précisément sur le projet Android Open Source Project de Google, mais avec des fonctionnalités de confidentialité et de sécurité supplémentaires.
Par exemple, CopperheadOS impose le chiffrage complet de la mémoire.

Eelo.io

Eelo est un OS open-source pour smartphone qui ne se base pas sur Android.
Il est destiné au grand public, et met l’accent sur la protection des données personnelles.
Il n’est pas encore viable et requiert un financement participatif.

Plasma

Basé sur Linux, Plasma a été conçu pour proposer une convergence entre Linux sur PC et Plasma Mobile sur smartphone et tablette.
Etant encore un prototype, il n’est pas encore possible de l’utiliser avec toutes ses fonctionnalités, mais on peut le tester.
Il n’est pas basé sur Android.

Librem 5

Librem 5 est un projet d’OS pour smartphone avec un système d’exploitation sécurisé et respectueux de la vie privée des utilisateurs.
Il devrait faire tourner différentes distributions Linux et proposer un mode desktop

Cependant, il est encore en développement, à la suite d’un financement participatif réussit.

BONUS !

F-Droid

F-Droid est un magasin d’applications libre pour les appareils Android.
Toutes les applications proposées sont libres et open-source et l’installation peut être directement faite depuis l’application.
Il est aussi possible de télécharger les fichiers « apk » depuis le site web des développeurs des applications et d’ajouter vos propres dépôts.

F-Droid n’étant pas disponible dans le Play Store, il doit être téléchargé via le site web et installé en autorisant l’installation depuis une source inconnue.

Aptdoide

Aptoide est un catalogue d’applications alternatif pour les appareil Android.
Celui-ci n’est pas géré par une autorité centralisée mais chaque utilisateur gère leur propre magasin.
On y retrouve donc aussi bien des applications libres que propriétaires.
Il y a plus d’applications que dans le Play Store et n’y est d’ailleurs pas référencé. Il faudra l’installer vous-même.

Pour cela,il faut télécharger le fichier « apk » disponible sur le site web officiel et l’installer en pensant à autoriser les applications venant de sources inconnues.
